## 1. 사용자 플로우

### 1.1 신규 사용자 온보딩 플로우

```
┌─────────────────────────────────────────────────────────────────────────┐
│                           ONBOARDING FLOW                                │
└─────────────────────────────────────────────────────────────────────────┘

[Landing Page]
     │
     ├──→ [Sign Up] ──→ [Email Verification]
     │         │
     │         ├── Email/Password
     │         ├── Google OAuth
     │         └── Kakao OAuth
     │
     ▼
[Onboarding Step 1: Welcome]
  "InsightFlow에 오신 것을 환영합니다"
  - 서비스 소개 (3초 애니메이션)
  - [시작하기] 버튼
     │
     ▼
[Onboarding Step 2: 첫 워크스페이스]
  "팀 또는 프로젝트명을 입력하세요"
  - 워크스페이스 이름 입력
  - [다음] 버튼
     │
     ▼
[Onboarding Step 3: 첫 분석 체험]
  "샘플 데이터로 체험하거나 파일을 업로드하세요"
  - [샘플 데이터로 체험] 버튼 (추천)
  - [내 파일 업로드] 버튼
     │
     ├──→ 샘플 데이터 선택 시
     │         │
     │         ▼
     │    [분석 진행 중...]
     │    (미리 계산된 결과 빠르게 로드)
     │
     └──→ 내 파일 업로드 시
               │
               ▼
          [파일 업로드]
               │
               ▼
          [분석 진행 중...]
               │
     ▼
[첫 대시보드 결과]
  - KPI 카드 하이라이트
  - 인사이트 카드 애니메이션
  - "이런 인사이트를 발견했어요!" 툴팁
     │
     ▼
[Dashboard Home]
  - 온보딩 완료!
```

## 5. 접근성 (A11y) 가이드

### 5.1 키보드 네비게이션
- Tab: 다음 포커스 가능 요소로 이동
- Shift+Tab: 이전 포커스 가능 요소로 이동
- Enter/Space: 버튼/링크 활성화
- Escape: 모달/드롭다운 닫기
- Arrow keys: 메뉴/리스트 탐색

### 5.2 스크린 리더
- 모든 이미지에 alt 텍스트
- 아이콘 버튼에 aria-label
- 동적 콘텐츠에 aria-live
- 폼 요소에 적절한 label

### 5.3 색상 대비
- 텍스트: 최소 4.5:1 대비
- 대형 텍스트: 최소 3:1 대비
- 인터랙티브 요소: 명확한 포커스 표시

### 5.4 모바일 접근성
- 터치 타겟: 최소 44x44px
- 제스처 대안: 버튼 제공
- 확대 지원: 최대 200%
