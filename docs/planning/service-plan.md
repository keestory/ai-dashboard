# AI Dashboard 서비스 기획서

## 1. 서비스 개요

### 1.1 서비스명
**InsightFlow** - AI 기반 데이터 분석 대시보드

### 1.2 슬로건
"Upload. Analyze. Act." / "업로드하면, 인사이트가 흐른다"

### 1.3 서비스 정의
Excel/CSV 파일을 업로드하면 AI가 자동으로 분석하여 인사이트와 액션을 제안하는 올인원 대시보드 서비스

### 1.4 핵심 가치
1. **즉시성**: 업로드 후 1분 내 인사이트
2. **접근성**: 비전문가도 프로 수준 분석
3. **실행력**: 분석에서 액션까지 원스톱
4. **이동성**: 웹과 앱 어디서나

## 3. 핵심 기능

### 3.1 파일 업로드 & 처리

#### 기능 상세
- **지원 형식**: .xls, .xlsx, .csv
- **업로드 방식**: 드래그앤드롭, 파일 선택, 클라우드 연동 (향후)
- **대용량 처리**: 청크 업로드 (100MB까지)
- **유효성 검증**: 파일 형식, 크기, 데이터 구조 검사
- **히스토리**: 이전 업로드 파일 관리

#### 사용자 플로우
1. 대시보드 화면에서 "새 분석" 클릭
2. 파일 드래그앤드롭 또는 선택
3. 업로드 진행률 표시
4. 데이터 미리보기 표시
5. "분석 시작" 클릭

### 3.2 자동 데이터 분석

#### 분석 유형
| 분석 유형 | 설명 |
|----------|------|
| 기술 통계 | 평균, 중앙값, 표준편차, 분포 |
| 트렌드 분석 | 시계열 데이터 추세 파악 |
| 이상치 탐지 | 비정상 데이터 포인트 발견 |
| 상관관계 | 변수 간 관계 분석 |
| 세그먼트 분석 | 그룹별 비교 분석 |
| 예측 분석 | 미래 값 예측 (Pro 이상) |

#### 처리 과정
1. 데이터 구조 자동 인식
2. 데이터 타입 추론 (숫자, 날짜, 카테고리)
3. 결측치/이상치 처리
4. 적합한 분석 방법 자동 선택
5. AI 모델 적용

### 3.3 AI 인사이트

#### 인사이트 유형
1. **핵심 발견**: 가장 중요한 데이터 포인트
2. **트렌드 알림**: 상승/하락 추세
3. **이상치 경고**: 비정상적인 변화
4. **패턴 발견**: 반복되는 패턴
5. **비교 분석**: 기간/그룹 비교

#### AI 프롬프트 전략
- Claude API 또는 OpenAI API 활용
- 데이터 컨텍스트 + 분석 결과 → 자연어 인사이트
- 비즈니스 친화적 언어 사용
- 수치와 함께 의미 설명

#### 인사이트 카드 구성
```
+----------------------------------+
| [아이콘] 카테고리               |
|----------------------------------|
| 핵심 메시지 (한 줄)              |
|                                  |
| 상세 설명 (2-3줄)                |
| - 수치 데이터                    |
| - 비교 정보                      |
|                                  |
| [관련 차트 썸네일]               |
|                                  |
| [자세히 보기] [액션 보기]        |
+----------------------------------+
```

### 3.4 대시보드 시각화

#### 차트 유형
| 차트 | 용도 |
|------|------|
| 라인 차트 | 시계열 트렌드 |
| 바 차트 | 카테고리 비교 |
| 파이 차트 | 구성 비율 |
| 히트맵 | 상관관계 |
| 스캐터 플롯 | 분포/군집 |
| KPI 카드 | 핵심 지표 |

#### 대시보드 레이아웃
- 드래그앤드롭 위젯 배치
- 반응형 그리드 시스템
- 프리셋 레이아웃 제공
- 저장 및 불러오기

#### 인터랙션
- 차트 확대/축소
- 데이터 포인트 호버 상세
- 필터링 및 드릴다운
- 기간 선택

### 3.5 액션 추천

#### 추천 유형
1. **즉시 실행**: 지금 바로 해야 할 것
2. **모니터링**: 지켜봐야 할 것
3. **조사 필요**: 더 분석이 필요한 것
4. **기회 포착**: 활용할 수 있는 것

#### 액션 카드 구성
```
+----------------------------------+
| [우선순위 배지] 액션 제목        |
|----------------------------------|
| 배경 설명                        |
| - 관련 인사이트 링크             |
|                                  |
| 권장 행동:                       |
| 1. 첫 번째 단계                  |
| 2. 두 번째 단계                  |
|                                  |
| 예상 효과: [수치]                |
|                                  |
| [완료] [나중에] [무시]           |
+----------------------------------+
```

#### 액션 관리
- 체크리스트 형태 관리
- 리마인더 설정
- 완료 추적
- 효과 측정 (향후)

### 3.6 리포트

#### 리포트 유형
1. **요약 리포트**: 핵심 인사이트 1페이지
2. **상세 리포트**: 전체 분석 결과
3. **비교 리포트**: 기간/세그먼트 비교
4. **커스텀 리포트**: 사용자 구성

#### 내보내기 형식
- PDF (프레젠테이션용)
- Excel (데이터 포함)
- 이미지 (차트별)
- 이메일 (링크 공유)

#### 리포트 템플릿
- 경영진 보고
- 마케팅 성과
- 재무 분석
- 커스텀 템플릿 저장

## 6. 기술 요구사항

### 6.1 성능 요구사항
- 파일 업로드: 100MB까지 60초 이내
- 분석 완료: 10MB 데이터 30초 이내
- 대시보드 로딩: 3초 이내
- 차트 인터랙션: 100ms 이내

### 6.2 보안 요구사항
- 파일 암호화 저장
- HTTPS 전송
- 인증 토큰 관리
- 사용자 데이터 격리
- GDPR/개인정보보호법 준수

### 6.3 확장성 요구사항
- 동시 사용자 1,000명 지원
- 일일 분석 10,000건 처리
- 저장 용량: 사용자당 10GB
