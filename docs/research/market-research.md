# AI Dashboard 시장 리서치 보고서

## 1. 시장 개요

### 1.1 시장 규모 및 성장
- BI(Business Intelligence) 시장은 2026년 기준 지속적인 성장세
- AI 기반 분석 도구 시장이 급성장 중
- 2025년 말까지 신규 BI 도구 구매의 40%가 AI 기반 분석 포함 전망

### 1.2 시장 점유율
| 플랫폼 | 시장 점유율 |
|--------|------------|
| Microsoft Power BI | 17.81% ~ 23.06% |
| Tableau | 13.74% ~ 15% |
| 기타 | 나머지 |

## 2. 주요 경쟁사 분석

### 2.1 Top-tier 플랫폼

#### Microsoft Power BI
- **강점**: Azure 통합, Copilot 자연어 쿼리, AutoML, 실시간 대시보드
- **가격**: 사용자당 월 $10~20
- **타겟**: Azure 생태계 기업

#### Tableau (Salesforce)
- **강점**: 최고 수준의 시각화, 직관적 데이터 탐색, 확장성
- **가격**: Creator $70/월, Explorer $35/월, Viewer $15/월
- **타겟**: 대기업, 데이터 스토리텔링 중시 조직

#### Looker (Google)
- **강점**: Gemini AI 예측 분석, 자연어 쿼리, Google Cloud 통합
- **타겟**: Google Cloud 사용 기업

#### Amazon QuickSight
- **강점**: AWS 통합, 서버리스, 사용량 기반 가격
- **타겟**: AWS 생태계 기업

### 2.2 AI-first 신흥 플랫폼

#### ThoughtSpot
- **특징**: 검색 기반 분석, Agentic AI
- **강점**: 구글처럼 검색하면 차트/테이블 반환
- **타겟**: 대규모 기업

#### Sisense
- **특징**: 임베디드 분석 전문
- **강점**: SaaS 제품 내 대시보드 임베딩
- **타겟**: SaaS 기업

#### Domo
- **특징**: 클라우드 기반 BI, 수백 개 데이터 소스 연결
- **강점**: 실시간 업데이트 대시보드

#### DataBrain
- **특징**: 임베디드 분석, 대화형 AI
- **강점**: 빠른 네이티브 임베딩, 정액제 가격

### 2.3 Excel 특화 AI 도구

#### Powerdrill Bloom
- **특징**: 2026년 표준, Raw data에서 경영진 요약까지
- **가격**: Free(3msg/일), Pro $15-20/월, Business $100/월
- **강점**: 기술적 마찰 없이 인사이트 도출

#### Ajelix
- **특징**: 20개 이상 AI 생산성 도구 + BI 플랫폼
- **가격**: Freemium (Free/Pro/Expert/Business)
- **강점**: 복잡한 Excel 작업 자동화, 인터랙티브 대시보드

#### Julius AI
- **특징**: AI 기반 통계 분석 및 데이터 사이언스
- **강점**: 다중 LLM, 자연어 프롬프트로 코드 실행

#### Sourcetable
- **특징**: AI 스프레드시트, Superagents
- **강점**: Google Analytics, Shopify, Stripe 등 연동

## 3. 2026년 주요 트렌드

### 3.1 AI 통합
- **자연어 쿼리**: SQL 대신 일반 영어로 질문
- **자동 데이터 준비**: 데이터 클렌징 자동화
- **스마트 시각화**: AI가 최적 차트 추천
- **이상치 탐지**: 자동 이상값 발견
- **예측 분석**: 미래 트렌드 예측
- **자동 인사이트 생성**: AI가 핵심 발견 요약

### 3.2 Conversational BI
- 정적 대시보드 시대 종말
- "질문하고 답변받기"가 새로운 표준
- 사전 구축된 리포트 클릭 불필요

### 3.3 Agentic AI
- "AI가 인사이트 보여주기" → "AI가 액션 실행"
- 문제 감지 시 자동으로:
  - 광고 예산 조정
  - 적절한 담당자에게 알림
  - 데이터 품질 이슈 수정

### 3.4 기타 트렌드
- 임베디드 분석 확대
- 윤리적 AI
- 비정형 데이터 시각화

## 4. 시장 기회 분석

### 4.1 Gap 분석
| 경쟁사 약점 | 우리의 기회 |
|------------|-----------|
| 복잡한 설정 | 원클릭 업로드 & 분석 |
| 높은 가격 | 합리적 가격 (SMB 타겟) |
| 기술 지식 필요 | 완전 자연어 인터페이스 |
| 대기업 중심 | 중소기업 친화적 |
| 설치형/복잡한 연동 | SaaS + 파일 업로드만으로 시작 |

### 4.2 차별화 포인트
1. **즉시 시작**: 회원가입 → 파일 업로드 → 인사이트 (3분 내)
2. **AI 액션 추천**: 단순 인사이트 넘어 실행 가능한 제안
3. **적정 가격**: SMB가 접근 가능한 가격대
4. **웹+앱**: 모바일 앱으로 언제 어디서나 확인
5. **한국어 최적화**: 한국 기업 데이터 특성 이해

### 4.3 타겟 시장
- **1차 타겟**: 데이터 분석가 없는 중소기업
- **2차 타겟**: 스타트업 운영팀/마케팅팀
- **3차 타겟**: 프리랜서/컨설턴트
