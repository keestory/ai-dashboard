## 2. 그로스 모델

### 2.1 성장 공식

```
MAU = (신규 유입 + 복귀) - 이탈
    = (Acquisition × Activation Rate) + Resurrection - Churn
```

### 2.2 레버리지 포인트

| 레버 | 현재 | 개선 가능성 | 우선순위 |
|------|------|------------|---------|
| Acquisition (유입) | X | 중 | P1 |
| Activation (활성화) | X% | 높음 | P0 |
| Retention (유지) | X% | 높음 | P0 |
| Referral (추천) | X | 높음 | P1 |
| Revenue (매출) | ₩X | 중 | P2 |

## 3. 채널 전략

### 3.1 채널 평가

| 채널 | CAC | 볼륨 | 확장성 | 우선순위 |
|------|-----|------|--------|---------|
| 오가닉 검색 (SEO) | 낮음 | 중 | 높음 | P0 |
| 콘텐츠 마케팅 | 중 | 중 | 높음 | P1 |
| 앱스토어 (ASO) | 낮음 | 중 | 중 | P0 |
| SNS (인스타/틱톡) | 중 | 높음 | 중 | P1 |
| 커뮤니티 | 낮음 | 낮음 | 낮음 | P2 |
| 유료 광고 | 높음 | 높음 | 높음 | P2 |
| 인플루언서 | 중 | 중 | 중 | P2 |

### 3.2 Bull's Eye Framework

```
        ┌─────────────────────┐
        │    What's Working   │
        │    (핵심 채널)       │
        │   ┌───────────────┐ │
        │   │   검증 중     │ │
        │   │  (테스트)    │ │
        │   │  ┌───────┐   │ │
        │   │  │가능성 │   │ │
        │   │  └───────┘   │ │
        │   └───────────────┘ │
        └─────────────────────┘
```

- **Inner Ring** (집중): SEO, ASO
- **Middle Ring** (테스트): 콘텐츠 마케팅, SNS
- **Outer Ring** (탐색): 유료 광고, 파트너십

## A/B 테스트 구현

### PostHog Feature Flags

```typescript
// apps/web/src/lib/experiments.ts
import { posthog } from './posthog'

export function getVariant(experimentKey: string): string | undefined {
  return posthog.getFeatureFlag(experimentKey) as string
}

export function isInExperiment(experimentKey: string, variant: string): boolean {
  return getVariant(experimentKey) === variant
}

// 사용
if (isInExperiment('onboarding_v2', 'test')) {
  // 새 온보딩 플로우
} else {
  // 기존 온보딩
}
```

### 이벤트 추적

```typescript
// 실험 노출 추적
posthog.capture('$experiment_started', {
  experiment: 'onboarding_v2',
  variant: getVariant('onboarding_v2'),
})

// 전환 추적
posthog.capture('onboarding_completed', {
  experiment: 'onboarding_v2',
  variant: getVariant('onboarding_v2'),
})
```
